# First Program in C++

## Syntax of C++ Program (Boilerplate Code)

```cpp
#include <iostream>
using namespace std;

int main(){
    return 0;
}
```

### Commands to run code in Terminal

```cpp
Command # 01 : g++ code.cpp (name of file)
Command # 02 : ./a.out
```

#### Shortcut of Command

```cpp
g++ code.cpp && ./a.out
```

## Example of a C++ Basic Code

```cpp
#include <iostream>
using namespace std;

int main(){
     cout << "C++ Basics" << endl;
    return 0;
}
```
